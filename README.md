# Radioamaterski izobraževalni portal

Next.js + Strapi spleta stran za izobraževanje radioamaterjev in kandidatov za radioamaterski izpit.

## Zaženi lokalno

```bash
yarn dev
```

## Environment variables

File: `.env.local`

```
STRAPI_URL=http://localhost:1337
```
